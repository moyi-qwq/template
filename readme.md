## 代码特性

+ 优先保证代码简洁和可读性，其次是常数
+ 数据结构使用指针（除了要卡空间的主席树）
+ 代码尽量用 `namespace` 封装
+ 轻度压行，不使用逗号压行
+ 使用 `template` 来复用代码
+ 代码符合 C++11 标准，且至少需要 C++11
+ 系统相关的部分只考虑 *unix


## 关于 PDF

+ 使用 pandoc 和 LaTeX 生成
+ 代码高亮使用 minted
